#  Superstore Sales Analysis Dashboard

## Project Overview

This project focuses on analyzing retail sales data from a Superstore dataset to extract meaningful business insights. It includes data cleaning, visualization, and reporting using Python and Power BI.

##  Objectives

* Perform data cleaning and preprocessing
* Analyze sales performance across regions, categories, and time
* Build an interactive dashboard for decision-making
* Generate insights to improve profitability

##  Tools & Technologies

* **Python** (Pandas, NumPy)
* **Power BI** (Dashboard & Visualization)
* **Excel/CSV Dataset**

##  Project Structure


├── Rohan_DATA_CLEANING.py        # Data preprocessing script
├── SuperStore_Sales_Dataset.csv  # Raw dataset
├── sales_dashboard.pbix         # Power BI dashboard
├── Sales_Report.pdf             # Final report
└── README.md

## Key Features

* Data cleaning and handling missing values
* Sales trend analysis (monthly/yearly)
* Region-wise and category-wise performance
* Profit vs Sales insights
* Interactive Power BI dashboard
